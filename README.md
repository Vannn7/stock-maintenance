# Stock Maintenance System

Inventory and stock maintenance system built using Google Apps Script, Google Sheets, and HTML.

## Overview

This application helps users manage inventory transactions, stock movement, and maintenance records through a web-based interface.

## Features

- User login
- Stock monitoring
- Product code lookup
- Inventory transaction recording
- Real-time data storage in Google Sheets
- Mobile-friendly web interface

## Technologies

- Google Apps Script
- Google Sheets
- HTML
- JavaScript
