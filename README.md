# Stock Maintenance System

Inventory Management System built using Google Apps Script, HTML, JavaScript and Google Sheets.

## Features

- Login multi user
- Role based access
- Purchasing admin access
- Stock In / Stock Out transaction
- Real-time stock validation
- Barcode support
- Auto item suggestion
- Dashboard analytics
- Stock opname module
- Procurement module
- Transaction history
- Mobile friendly interface

## Technology

- Google Apps Script
- Google Sheets
- HTML
- CSS
- JavaScript
- Chart.js

## Modules

### 1. Transaction Module
Record stock in and stock out transactions.

### 2. Stock Master
Manage inventory master data and current stock.

### 3. Stock Opname
Generate stock opname reports and inventory verification.

### 4. Procurement
Monitor inventory shortages and procurement requirements.

### 5. Dashboard
View stock movement statistics and transaction analytics.
